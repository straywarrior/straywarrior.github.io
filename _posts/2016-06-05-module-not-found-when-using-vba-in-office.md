## English Version

Both Windows and Office support multiple languages and someone like me may use different language settings for them. However, this can cause some strange problems. My operation system is Windows 8.1 Pro and it is set to display in English. I installed Simplified Chinese version of Office 2016 and it was set to display in Simplified Chinese (By default, it doesn't have English language pack). Recently, when I tried to use VBA (Visual Basic for Applications) in Excel, I met the error message "Module Not Found". I re-installed the Office 2016 and the problem was still here. Finally, I changed my system language and region settings to Simplified Chinese and China, then it worked. Note: The language, format and location must be matched. At first, I set the language of Office and Windows to Simplified Chinese but remained the format "English (United States)" and it didn't work.[![Office Language Settings](/assets/images/2-300x242.png)](/assets/images/2.png)[![Windows Region Settings](/assets/images/1-239x300.png)](/assets/images/1.png)

## 中文版本

Windows和Office都支持多语言，有些人（比如我）可能会为它们使用不同的语言设置。 然而，这有可能引起一些奇怪的问题。我的操作系统是Windows 8.1 Pro并且被设置为使用英语显示。我安装了简体中文版本的Office 2016并且让它以简体中文显示（默认情况下Office不带有英语语言包）。 最近，我试图在Excel中使用VBA（Visual Basic for Applications）时遇到了这样的错误“模块未找到”。我重新安装了Office但是问题并没有解决。 最终，我将系统语言改为简体中文，区域设置改为中国，这起了作用。 注意：语言、格式、地区必须是匹配的。一开始，我设置了Office和Windows 的语言为简体中文，但是格式为“英语（美国）”，导致没有解决问题。
